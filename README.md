# Projeto Fluxo Feliz

Este projeto foi desenvolvido como parte da atividade da disciplina, com o objetivo de praticar o fluxo completo de versionamento utilizando Git e GitHub.

## Etapas realizadas

### Etapa 1 — Projeto inicial
- Criação do projeto local
- Inicialização do Git
- Primeiro commit com estrutura básica
- Criação do repositório remoto no GitHub
- Conexão entre repositório local e remoto
- Push do projeto para a branch `main`

### Etapa 2 — Primeira solicitação
- Criação da branch `feature/solicitacao-1`
- Implementação de uma melhoria simples
- Commit e push da branch
- Abertura do Pull Request
- Merge da branch na `main`

### Etapa 3 — Segunda solicitação
- Criação da branch `feature/solicitacao-2`
- Melhoria visual no `index.html` com cores e estilo via CSS
- Commit e push da branch
- Abertura do Pull Request
- Merge da branch na `main`

### Etapa 4 — Documentação
- Criação da branch `feature/readme`
- Criação do arquivo `README.md` com histórico das etapas
- Commit e push da branch
- Abertura do Pull Request
- Merge da branch na `main`

## Estrutura do projeto

- `index.html`: Página principal com conteúdo e melhorias visuais
- `README.md`: Documentação do projeto e histórico das etapas
